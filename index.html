About this blog

Stay technical and approachable.

Built for engineers who want to write without ceremony. The layout is responsive, typography is tuned for code snippets, and everything lives in version control.

Workflow

Write in plain HTML under posts/, then list the entry in data/posts.js. No build step required.

Styling

Customize colors and spacing in assets/styles.css. Fonts come from Google Fonts for a distinctive feel.

Deploy

Push to GitHub, enable Pages for the repository, and set the source to the main branch root.
